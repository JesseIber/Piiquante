Pour pouvoir utiliser correctement l'API, veuillez suivre ces instructions :

- Clonez ce repository

Pour lancer le back de Piiquante naviguer à l'aide de l'invité de commande dans le dossier puis tapez : 
- 'npm install' pour installer les dépendances requises
- 'npm start' pour lancer le serveur backend

Pour le front de l'application :
- npm start
